# Images & Assets

This directory contains visual assets for the Command Center project.

## Contents

- `cc-usage-report.png` - Example usage report screenshot (displayed in main README)

## Guidelines

### Screenshots
- Format: PNG (preferred) or JPEG
- Resolution: High-quality, readable text
- Naming: Descriptive names with hyphens (e.g., `terminal-output-example.png`)

### Adding New Images

1. Place image files in this directory
2. Reference in documentation using relative paths:
   - From root: `![Description](assets/images/filename.png)`
   - From docs/chapters: `![Description](../../assets/images/filename.png)`
